## Features
- 🌍 **Multi-Language Support**: English, Français, Deutsch, العربية.
- 🔄 **Dynamic RTL Layout**: Full Right-To-Left layout mirroring for Arabic.
- ⚡ **Real-time Animations**: Smooth transitions and a dynamic degree progress bar.
- 📱 **Fully Responsive**: Optimized for desktop, tablet, and mobile devices.

## Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/EzzedineYassine/yassin-portfolio-v1.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd yassin-portfolio-v1
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npm run dev
   ```
